webdriverio-example
===================

A simple example of using `docker-selenium` and `webdriverio`

### Requirements
* `node >= 10`
* `docker`
* `docker-compose`

## Usage
```bash
# Install webdriver
git clone https://github.com/amilajack/webdriverio-example
cd webdriverio-example
yarn

# Run docker
docker-compose up
```
